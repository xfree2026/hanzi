# 汉字字帖生成器 — 产品需求文档（PRD）

## 1. 产品概述

一款面向书法学习者、家长、教师与传统文化爱好者的浏览器端汉字临摹字帖生成工具。内置《三字经》《百家姓》《千字文》《唐诗》《伤寒论》《黄帝内经》《针灸大成》等经典文本资源库，支持多种字格样式与版式，可在线预览、打印为标准 A4 字帖，并可一键为指定字/页面生成 AI 配图插图。

- 解决问题：现成字帖样式固定、文本有限、不可定制；用户希望按需生成可打印的字帖。
- 目标用户：儿童书法启蒙、中小学语文教师、中医典籍研读者、书法爱好者。
- 市场价值：免费、可扩展、即开即用，覆盖蒙学与中医经典两大特色文本。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 权限 |
|------|----------|------|
| 普通用户 | 直接打开网页 | 浏览资源、生成字帖、打印、生成 AI 插图 |

### 2.2 功能模块

1. **字帖工作台**：资源选择、文本编辑、字格样式、版式、网格规格、字体大小、AI 插图
2. **资源库浏览**：内置经典文本列表、查看原文、加入字帖
3. **预览与打印**：实时预览、A4 适配、打印设置、导出 PNG

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 字帖工作台 | 资源选择器 | 选择内置资源或自定义输入文本 |
| 字帖工作台 | 文本编辑器 | 显示/编辑待生成字帖的字符序列，过滤非汉字 |
| 字帖工作台 | 字格样式 | 田字格 / 米字格 / 九宫格 / 描红 / 轮廓 / 实心字模 |
| 字帖工作台 | 版式选择 | 横排左到右 / 竖排右到左 |
| 字帖工作台 | 网格规格 | 每行字数、每页字数、字格大小、字体 |
| 字帖工作台 | AI 插图 | 输入提示词为当前页生成配图，可插入页眉/页脚 |
| 字帖工作台 | 实时预览 | SVG 矢量渲染，所见即所得 |
| 字帖工作台 | 打印导出 | 调起浏览器打印、A4 纸张适配、可导出 PNG |
| 资源库 | 文本列表 | 按类别（蒙学/诗词/中医）列出全部内置文本 |
| 资源库 | 原文查看 | 弹窗展示完整原文，支持"加入字帖" |

## 3. 核心流程

用户打开页面 → 在左侧资源库选择文本（或粘贴自定义文本）→ 中间预览区显示当前字帖 → 在右侧设置面板调整字格/版式/网格/字体 → 点击"AI 配图"输入提示词为页面生成插图 → 点击"打印"调起浏览器打印对话框输出 A4 字帖。

```mermaid
    "E -- 是 --> F[输入提示词调用图像 API]"
    "F --> G[插图插入页面]"
    "G --> H[预览最终字帖]"
    "E -- 否 --> H"
    "H --> I[点击打印调起浏览器打印]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：东方书卷 · 现代克制。墨色 + 米白宣纸 + 朱砂红点缀 + 沉香金细节。
- **主色**：`#1a1714`（墨黑）/ `#f5efe1`（宣纸米白）/ `#c0392b`（朱砂红）/ `#b08d57`（沉香金）
- **字体**：标题用 `Noto Serif SC`（思源宋体）配 `Ma Shan Zheng`（马善政手书）做装饰；正文用 `Noto Sans SC`；字模默认 `STKaiti / KaiTi / 楷体`。
- **布局**：左侧固定资源库抽屉，中央预览画布（白纸+阴影），右侧设置面板；顶部细线条工具栏带品牌印章 logo。
- **细节**：宣纸纹理背景、印章式按钮、朱砂落款、雕版竖排标题。

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 字帖工作台 | 顶部工具栏 | 印章 logo、标题、打印按钮、导出按钮、AI 配图按钮 |
| 字帖工作台 | 左侧资源库 | 分类标签（蒙学/诗词/中医）、文本卡片列表、搜索框 |
| 字帖工作台 | 中央预览 | A4 白纸 SVG、页码、滚动多页、当前页高亮 |
| 字帖工作台 | 右侧设置 | 字格样式卡片、版式切换、网格规格滑块、字体下拉 |
| 字帖工作台 | AI 配图弹窗 | 提示词输入、风格选择、生成按钮、预览缩略图、插入位置 |
| 资源库弹窗 | 原文展示 | 竖排原文阅读、加入字帖按钮、关闭按钮 |

### 4.3 响应式

桌面优先（1280px+），平板自适应折叠侧栏，移动端仅保留预览与基础设置（字帖工具本质上是桌面打印场景）。

### 4.4 AI 插图设计指引

- 风格预设：水墨写意、工笔重彩、线描白描、版画木刻 4 种
- 调用图像生成 API（默认 TRAE 内置 `text_to_image`，可配置外部 URL）
- 单次生成 1 张，可重新生成，可插入到字帖页眉或单独作为扉页
